# 🌍 世界地图挑战 (World Map Challenge)

一个基于浏览器的世界地图地理知识游戏，使用 Leaflet 和 GeoJSON 实现交互式地图问答。

---

## 🎮 游戏模式

| 模式 | 玩法 | 说明 |
|------|------|------|
| **📍 点名找国** | 给出国家名称 → 在地图上点击该国位置 | 核心模式，考验地理定位能力 |
| **🏙️ 首都找国** | 给出首都名称 → 在地图上点击对应国家 | 结合首都知识 |
| **🗺️ 轮廓猜名** | 仅显示国家轮廓（其余区域变暗）→ 输入国家名称 | 高难度，考验对国家形状的认知 |
| **🎯 高亮猜名** | 地图高亮一个国家 → 输入该国名称 | 适合新手入门 |
| **⏱ 限时挑战** | 上述模式 + 倒计时，尽可能多得分 | 60-120秒限时 |

## 🏋️ 练习模式

练习模式下**不扣分、不限次数**，答错时提供渐进式提示：

| 错误次数 | 提示内容 |
|---------|---------|
| 第1次 | "再试一次！" |
| 第2次 | 显示目标国家所在大洲 |
| 第3次 | 显示国家名称首字母 |
| 第4次 | 地图自动缩放到目标区域并闪烁 |
| 第5次 | 直接显示正确答案，标记为"未掌握" |

## 📊 计分系统

```
最终得分 = (基础分100 + 速度加成 + 连击加成 - 提示扣分) × 难度系数
```

- **速度加成**：5秒内 +50，10秒内 +30
- **连击加成**：连续答对每级 +10×连击数
- **难度系数**：简单 ×1.0 / 中等 ×1.3 / 困难 ×1.5
- **小国加成**：面积排名 >40 的国家得分 ×1.5

## 🎯 难度分级

| 难度 | 国家池 | 说明 |
|------|--------|------|
| 🟢 简单 | ~40个大国 | 面积前40名，排除南极洲 |
| 🟡 中等 | ~100个国家 | 加入欧洲/亚洲中型国家 |
| 🔴 困难 | ~170个国家 | 包含小型岛国和内陆小国 |

## 🌐 语言支持

- **中文 / English** 双语界面，一键切换（顶部栏按钮或 `Ctrl+L`）
- 国家名称、首都、UI 文本均支持双语
- 语言偏好自动保存

## 🔤 辅助输入

在"轮廓猜名"和"高亮猜名"模式下，输入框支持智能模糊搜索：

- 中文包含匹配（输入"加" → 加拿大、加纳、加蓬…）
- 英文前缀/包含匹配（输入"Ca" → Canada, Cambodia, Cameroon…）
- 键盘 ↑↓ 选择候选，Enter 确认

## 🎨 视觉特性

- **暗色主题** UI，深蓝海洋底色
- **12色调色板**，邻国自动分色（贪心图着色算法）
- **灰色虚线** 标注争议地区（索马里兰、北塞浦路斯）
- **悬停高亮** + **点击反馈** 动画
- 响应式布局，支持桌面和移动端

## 🚀 运行方式

### 方法一：Python HTTP 服务器（推荐）

```bash
cd world-map-game
python -m http.server 8080
```

浏览器打开 `http://localhost:8080`

### 方法二：任意 HTTP 服务器

由于项目使用了 `fetch` 加载 GeoJSON 数据，需要 HTTP 服务器环境，不能直接双击 HTML 文件打开。可以使用：

- VS Code Live Server 插件
- Node.js `npx serve .`
- 或任意静态文件服务器

### 无外部依赖

项目**不需要 npm install**，Leaflet 通过 CDN 引入，其他均为原生 JavaScript。

## 🗺️ 数据来源与处理

### 原始数据

地图几何数据来自 **Natural Earth** 1:110m 文化矢量数据集（版本 4.1.0），经 `world-geo-json-zh-main` 项目转换为含中文名称的 GeoJSON。

### 预处理操作

`scripts/preprocess.py` 对原始数据执行了以下处理：

1. **台湾标注修正**：将台湾（TW）几何体合并到中华人民共和国（CN）中
2. **中印边界修正**：修正藏南地区和阿克赛钦地区的边界表示
3. **争议地区标注**：索马里兰、北塞浦路斯等未被普遍承认的实体标记为"争议"，以灰色虚线显示
4. **元数据补充**：添加英文名称、大洲归属（含中英文）、首都名称（含中英文）
5. **邻国图着色**：使用 Shapely 构建邻接图 + 贪心着色算法（Welsh-Powell），确保邻国颜色不同
6. **面积排序**：计算每个实体的面积用于难度分级

## ⚠️ 重要声明

### 数据来源

本地图使用的原始几何数据来自 **Natural Earth**（美国开源项目），其边界表示基于**事实控制线（de facto）**而非任何国家的主张线（de jure）。该数据集在以下方面可能与各国官方立场存在差异：

- **南海诸岛与九段线**：当前数据**不包含**南海九段线及南海诸岛的详细表示
- **中印边界**：预处理脚本已对藏南和阿克赛钦地区进行了边界修正，但受限于原始数据精度（1:110m），边界精度有限
- **克什米尔地区**：争议区域的表示可能不完整
- **数据精度**：1:110m 比例尺意味着 1cm ≈ 1100km，边界为高度简化的概略表示，**不可用于精确测量或法律用途**

### 争议地区处理

以下未被普遍承认的实体在地图上以**灰色虚线**标注，不参与游戏题目生成：

- 索马里兰（Somaliland）
- 北塞浦路斯土耳其共和国（Northern Cyprus）

### 免责

本游戏仅供教育和娱乐目的，地图边界和领土表示不应被视为任何政治立场或法律主张。如需精确的地图数据，请参考各国官方测绘机构发布的权威数据。
